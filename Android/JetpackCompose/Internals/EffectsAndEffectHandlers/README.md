# 6. 이펙트 및 이펙트 핸들러 (Effects and effect handlers)

이펙트 핸들러에 대해 알아보기 전에, 먼저 `사이드 이펙트`에 대해 간략하게 살펴보도록 합니다. 이는 Composable 트리에서 사이드 이펙트를 제어하는 것이 왜 중요한지 몇가지 맥락을 제공합니다.
